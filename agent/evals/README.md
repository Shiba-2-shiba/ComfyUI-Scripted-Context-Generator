# Evals (Common)

目的: Skillの出力が期待通りかを最小コストで確認する。

## Success Criteria
- Outcome: 必要なファイルや成果が生成される
- Process: 手順が守られる
- Style: 指定フォーマットが崩れない

## Minimal Eval Prompts
Use `evals/prompts/` and check:
- Structure: required sections present
- Constraints: forbidden items absent
- Clarity: actionable output
