# Naming Conventions (Common)

## Files
- Use `snake_case` for scripts, data files, and documentation pages.
- Exceptions (UPPER_CASE): `README.md`, `AGENT_GUIDE.md`, `SKILL.md` — these are convention-level
  entrypoints intended to be immediately recognizable.
- Use `README.md` for folder entrypoints.

## Folders
- Use descriptive nouns, avoid abbreviations.
- Keep depth <= 3 where possible.

## Logs
- Use `*_log.md` for human-readable logs.
